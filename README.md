A simple, responsive product listing page built with plain HTML and CSS. It demonstrates a clean product grid layout with product cards, hover states, and straightforward semantic markup—no JavaScript required.

Features ✨
Responsive grid for product cards (mobile → desktop)
Clean, reusable CSS in products.css
Accessible semantic HTML structure in products.html
Easy to customize styles (colors, spacing, grid columns)
